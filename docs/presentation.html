The Problem & Solution

The Problem

German Kitas rely on WhatsApp, paper forms, and phone calls for daily communication and child documentation — none of which are GDPR-compliant for child data.

Pain Points

  • ❌WhatsApp groups — not GDPR-compliant for child data
  • ❌Paper observation forms — no searchability, easily lost
  • ❌Phone calls — no audit trail, no documentation
  • ❌Educator hours wasted on manual paperwork
  • ❌No structured child development tracking

The Solution

A unified, role-based portal for parents, educators, and management with automated workflows, AI-assisted documentation, and multi-channel communication.

What Kita Connect delivers

  • ✅GDPR-compliant by design — EU-only infrastructure
  • ✅Structured child development documentation
  • ✅AI-generated learning story drafts (Lerngeschichten)
  • ✅Automated notifications: In-App, Email, SMS
  • ✅Near-zero monthly cost (~0 € on free tiers)

GDPR Pseudonymization

Real child names never leave the EU database. Only the pseudonym is sent to the Anthropic API.

How it works
  • 1️⃣Every child gets a stable pseudonym at creation (e.g. Kind-A1B2) stored in ai_pseudonym_map
  • 2️⃣n8n fetches the pseudonym — never the real name
  • 3️⃣Prompt sent to Claude: "Kind-A1B2 (4 Jahre) hat heute beim Malen..."
  • 4️⃣Claude writes the story using the pseudonym
  • 5️⃣Educator replaces pseudonym with real name before publishing

⚠️ Anthropic DPA

Anthropic is a US company. Pseudonymization is the primary technical protection. A DPA + Standard Contractual Clauses (SCCs) must be signed before production launch.

Design Principles

Cost Optimization

Only the first time a question is asked does it hit the Claude API. Every repeat question is served from the faq_cache table for free. The hit_count field tracks which questions are most popular.

GDPR Safe

No personal data is included in FAQ prompts. Claude is only told the question text — never the user's name, child's name, or any identifying information.

Scope Limiting

The system prompt restricts Claude to general Kita topics only. If a question is too specific (e.g. exact opening hours), Claude replies: "Bitte wende dich direkt ans Kita-Team."
